raspbygopro
===========

Convert your raspberry pi in one Go Pro

## Requeriments
- Raspberry Pi A/B/B+
- Raspberry pi camera
- Switch
- Coffee or beer

##Install service
When raspberry pi run the service 'raspbigopro' run automatic
``` shell
sudo cp raspigopro /etc/init.d/
chmod -x /etc/init.d/raspigopro
sudo update-rc.d raspigopro defaults
```
